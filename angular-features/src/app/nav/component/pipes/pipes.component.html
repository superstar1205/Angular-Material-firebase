Built-in Angular Pipes
  • Uppercase Pipe
    {{title | uppercase}}
  • Lowercase Pipe
    {{title | lowercase}}
  • TitleCase Pipe
    {{title | titlecase}}
  • Currency Pipe
    {{6589.23 | currency:'INR': 'symbol'}}
  • Date pipe
    {{today_date | date:'d/M/y'}}{{today_date | date:'shortTime'}}
  • Decimal Pipe
    {{ 454.78787814 | number: '3.4-4' }}
  • Json Pipe
    {{ json_val | json }}
  • Percent Pipe
    {{00.54565 | percent}}
  • Slice Pipe
    {{months | slice:2:6}}
  • Async Pipe
    {{ count$ | async }}
    {{ message }}
  • Custom Pipe
    {{2 | custom: 5}}
