# ng2-ShoppingCart
Shopping Cart implementation using angular 2

***Installation***

```
git clone git@github.com:bsubedi26/ng2-ShoppingCart.git
cd ng2-ShoppingCart
npm install
No server: npm run start
Server Running: npm run server (change proxy settings in proxy.conf.json file)
```
